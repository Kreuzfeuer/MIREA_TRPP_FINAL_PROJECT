# MIREA_TRPP_FINAL_PROJECT
##Application description
This is the usual note-taking application for the book lover.

##Dependencies
The application uses: 
    1) Spring Boot;
    2) Spring Security;
    3) Spring Data JPA;
    4) Spring MVC;
    5) Spring Validation;
    6) Project Lambok;
    7) Postgresql.

##Intial command
    mvn spring-boot:run
